# Edge Deployment Tier

Synthetic evaluation fixture. Confusable with the cloud deployment tier.

## Availability

The edge tier runs fully on-device. It requires macOS 13 or later and 8
gigabytes of unified memory. No network connection is required at inference time.

## Data Residency

On the edge tier, all audio and transcripts stay on the local machine.
Nothing is transmitted off-device unless the user explicitly exports a note.

## Update Cadence

Edge models update through the app store review process, roughly every six weeks.
