# Cloud Deployment Tier

Synthetic evaluation fixture. Confusable with the edge deployment tier.

## Availability

The cloud tier runs in a managed data center. It requires an active network
connection and an API key. Clients need no local GPU.

## Data Residency

On the cloud tier, audio is processed in region and deleted within
twenty-four hours. Transcripts are retained only if the customer opts into history.

## Update Cadence

Cloud models update continuously through a rolling deployment, often several
times per week.
